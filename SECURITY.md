# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| v0.1.x  | ✅ |

## Reporting a vulnerability

If you discover a security vulnerability in NeoDLP, please report it privately.

**Do not open a public issue.** Instead, email **rkriad585@gmail.com** with:

- A description of the issue
- Steps to reproduce
- Potential impact

You will receive a response within 48 hours. We'll work with you to understand
and fix the issue promptly.

## Scope

This policy covers the NeoDLP codebase and its official releases. It does not
cover third-party dependencies — report those to their respective maintainers.
